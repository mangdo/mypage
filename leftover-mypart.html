1) 판매글의 CRUD

[ CBV를 통한 요청처리 ]
: 제네릭 뷰를 상속받으면 반복되는 코드를 없애서 간결한 코드를 작성할 수 있다는 장점때문에 CBV를 사용하였습니다.
: 이를 통해 하나의 클래스에서 여러 HTTP Method를 처리합니다.

[ 판매글의 조회 ]
: queryset.filter를 이용하여 검색합니다.
: 이를 이용하여 자신의 동네에 있는 판매글을 검색합니다.
: pagination을 오버라이딩하여 만든 custompagination을 이용하여 페이징 처리를 합니다.

2) 판매글의 조회

: CBV를 통해 요청처리를 합니다.
: queryset.filter를 이용하여 자신의 동네에 있는 음식점을 검색합니다.
: pagination을 오버라이딩하여 만든 custompagination을 이용하여 페이징 처리를 합니다.

[ 음식점의 개별페이지 ]
